Getting Started

What is HEMA?

HEMA (Healthcare Evaluation and Medical Analysis) is a comprehensive web-based platform designed for medical professionals to analyze and evaluate medical imaging data, 3D models, and related documentation.

System Requirements
  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • JavaScript enabled
  • Minimum 4GB RAM recommended
  • Good internet connection for loading resources
Quick Start
  1. Open HEMA in your web browser
  2. Choose your analysis mode (Segmentation, 3D, or Text)
  3. Upload your medical files using the file input
  4. Wait for the viewer to initialize
  5. Start analyzing and annotating your data

Segmentation Mode

Segmentation mode allows you to analyze medical images slice by slice, perfect for reviewing MRI, CT scans, and other volumetric data.

Supported File Formats
NRRD Files (.nrrd)

Near Raw Raster Data format, commonly used for medical imaging

NIfTI Files (.nii, .nii.gz)

Neuroimaging Informatics Technology Initiative format

Navigation Controls
Control Action
← / → Navigate between slices
Mouse Wheel Scroll through slices
Slice selector Jump to specific slice

3D Model Mode

3D mode enables visualization and analysis of three-dimensional medical models and anatomical structures.

Supported Formats
STL Files (.stl) - Stereolithography format for 3D models
3D Controls
Control Action
Left Click + Drag Rotate model
Mouse Wheel Zoom in/out
Right Click + Drag Pan model

Text Analysis Mode

Text analysis mode provides tools for reviewing and analyzing medical reports, documentation, and text-based data.

Features
  • Text file upload and display
  • Search functionality
  • Annotation support
  • Export capabilities

Drawing Annotations

HEMA includes a powerful annotation system that allows you to draw directly on medical images and save your annotations locally in your browser.

Drawing Tools
Pen Tool

Draw freehand annotations

Eraser Tool

Remove unwanted annotations

How to Use Annotations
  1. Load your medical image in Segmentation mode
  2. Click the Pen tool to activate drawing mode
  3. Click and drag on the image to draw
  4. Use the Eraser tool to remove unwanted marks
  5. Annotations are automatically saved to your browser
Note: Annotations are stored locally in your browser and will persist between sessions. They are associated with specific files and slices.

Comments System

Add detailed textual observations and notes to specific slices of your medical images.

Adding Comments
  1. Navigate to the slice you want to comment on
  2. Locate the comments panel on the right side
  3. Type your observation or note in the text area
  4. Click Add Comment to save
Managing Comments
  • Comments are automatically timestamped
  • Multiple comments per slice are supported
  • Comments persist between browser sessions
  • Individual comments can be deleted

Annotation Database

The database view provides a comprehensive overview of all your saved annotations and comments across all analyzed files.

Accessing the Database

Click the Database button in the top navigation bar to open the annotation database in a new tab.

Database Features
  • View all files with annotations/comments
  • See statistics for each file (slices, annotations, comments)
  • Detailed view of individual file data
  • Organized by slice for easy navigation
  • Delete functionality for data management
Storage Location: All data is stored locally in your browser using IndexedDB. Data will not be accessible from other browsers or devices.

Export Functionality

Export your annotation and comment data for backup, sharing, or further analysis.

Export Options
JSON Export

Complete data export in JSON format including all annotations, comments, timestamps, and metadata.

PDF Report (Coming Soon)

Formatted report with summaries and visualizations of your analysis data.

How to Export
  1. Click the Export button in the main navigation
  2. Or use the export function in the Database view
  3. Choose your export format
  4. The file will be automatically downloaded

Troubleshooting

Common Issues

Solutions:
  • Ensure you've selected the correct analysis mode before uploading
  • Check that your file format is supported
  • Try refreshing the page and re-uploading
  • Check the browser console for error messages

Solutions:
  • Ensure cookies and local storage are enabled
  • Check if you're in private/incognito browsing mode
  • Verify you have sufficient disk space
  • Try clearing browser cache and reload

Solutions:
  • Close other browser tabs to free up memory
  • Use a computer with more RAM if possible
  • Consider processing files in smaller batches
  • Update to the latest browser version
Getting Help

If you continue experiencing issues:

  • Check your browser's developer console for error messages
  • Try using a different browser
  • Ensure your browser supports modern web standards
  • Contact your IT support team for assistance
